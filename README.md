# 🏠 House Price Prediction using Linear Regression

## Overview
This project is extremely beginner friendly that...predicts house prices using the Linear Regression algorithm.

## Dataset
The dataset contains the following features:

- Area
- Bedrooms
- Bathrooms
- Stories
- Main Road
- Guest Room
- Basement
- Hot Water Heating
- Air Conditioning
- Parking
- Preferred Area
- Furnishing Status

**Target Variable:** Price

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Workflow
1. Data Loading
2. Data Inspection
3. Data Preprocessing
4. Encoding Categorical Features
5. Train-Test Split
6. Model Training
7. Prediction
8. Model Evaluation

## Results
- MAE: 979679.69
- RMSE: 1331071.42
- R² Score: 0.6495
